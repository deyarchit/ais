# Quick Task 260323-q65: Make it optional to show references in terminal output — Summary

**One-liner:** Added `--no-refs` CLI flag that suppresses the Sources citation block in both one-shot and REPL modes by threading a `showRefs bool` through entry points.

## What Was Built

A single boolean flag `--no-refs` accepted by `ais`. When present, `render.Sources(...)` is not called in either one-shot (`runOneShot`) or REPL (`repl.Run`) mode. Default behavior (Sources printed) is fully preserved when the flag is absent.
